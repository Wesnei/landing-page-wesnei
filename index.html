Drip Store

O eCommerce Drip Store oferece uma plataforma intuitiva com uma página inicial destacando promoções, categorias de produtos e produtos populares. Os usuários podem navegar facilmente por categorias, filtrar produtos e concluir compras com um processo de checkout simples. Gerenciamento de cursos

Este é um projeto CRUD desenvolvido com FastAPI, SQLAlchemy e PostgreSQL, permitindo a criação, leitura, atualização e exclusão de dados de forma eficiente. A aplicação utiliza FastAPI para rotas rápidas e seguras, SQLAlchemy para a interação com o banco de dados e PostgreSQL como sistema de gerenciamento de banco de dados relacional. O projeto oferece uma API robusta e escalável para manipulação de dados em tempo real.

Horizontes Travel

Este é um projeto de uma agência de viagens desenvolvido em Java, utilizando JavaFX para criar a interface gráfica e Scene Builder para facilitar a construção do layout da aplicação. O sistema conecta-se a um banco de dados MySQL, onde são armazenadas informações cruciais sobre reservas, clientes, pacotes e destinos. A aplicação permite que usuários e agentes de viagens realizem reservas de pacotes de forma eficiente e interativa.

Jota Barber

A landing page da Jota Barber é uma página simples e atrativa, focada em apresentar os serviços oferecidos pela barbearia, como cortes de cabelo e barba. Destaca imagens do ambiente e dos serviços. A página também conta com informações de contato e localização, tornando fácil para os visitantes agendarem seu próximo corte.
